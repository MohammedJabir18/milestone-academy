## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/MohammedJabir18/milestone-trading-academy.git

# Navigate to project directory
cd milestone-trading-academy

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
